# sumer

`sumer` is bash AST parser written in JavaScript.

## Credits

Heavily based on [`acorn`](https://github.com/ternjs/acorn).

## Install

```sh
$ npm install sumer
```

## Usage

```js
import sumer from 'sumer'
```
